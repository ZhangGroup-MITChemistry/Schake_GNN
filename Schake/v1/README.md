# Schake (version 1) 

This architecture is detailed in the following work:
* Airas, J.; Zhang, B. Scaling Graph Neural Networks to Large Proteins. *J. Chem. Theory Comput.* **2025**, *21*(4), 2055-2066. [JCTC paper](https://pubs.acs.org/doi/full/10.1021/acs.jctc.4c01420)

An example of how to use the Schake architecture is shown in the provided [Jupyter Notebook](Example_create_Schake.ipynb). To use Schake, the following packages will be needed:
* PyTorch [link](https://pytorch.org/)
* PyTorch-Geometric, with torch_scatter and torch_cluster [link](https://pytorch-geometric.readthedocs.io/en/latest/install/installation.html)
